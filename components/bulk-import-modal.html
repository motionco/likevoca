대량 개념 추가

파일 형식 안내

다음 형식의 파일을 업로드할 수 있습니다:

  • CSV 파일: 다국어 개념 데이터가 포함된 CSV 파일
  • JSON 파일: 다국어 개념 데이터가 포함된 JSON 파일

파일에 도메인이 지정되지 않은 경우 사용됩니다.

파일에 카테고리가 지정되지 않은 경우 사용됩니다.

JSON 설정

JSON 파일은 다음 형식을 따라야 합니다:

    "domain": "daily",
    "category": "greeting",
    "expressions": {

      },
      "english": {
        "word": "hello",
        "pronunciation": "həˈloʊ",

  }
]
