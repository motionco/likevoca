자동 업데이트됨 ✅

📋 현재 페이지 메타태그 정보

URL을 입력하고 '분석하기'를 클릭하세요.

📖 사용법

1.

위의 URL 입력창에 확인하고 싶은 LikeVoca 페이지 URL을 입력합니다.

2.

'분석하기' 버튼을 클릭하여 페이지의 메타태그 정보를 확인합니다.

3.

원하는 소셜 미디어 플랫폼의 디버거 도구 버튼을 클릭합니다.

4.

각 플랫폼의 디버거에서 'Scrape Again' 또는 '다시 크롤링' 버튼을 클릭합니다.
