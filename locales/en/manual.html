Getting Started

1. Account Creation

  1. Visit the LikeVoca homepage
  2. Click the "Sign Up" button
  3. Choose email registration or social login
  4. Fill in required information and agree to terms
  5. Complete email verification

2. First Login & Setup

Complete these settings on your first login:

  • Select target learning language
  • Set current language level
  • Set learning goals
  • Complete basic settings

2. Creating Your Own Vocabulary

  1. Click "My Vocabulary" on the dashboard
  2. Select "Create New Vocabulary"
  3. Enter vocabulary name and description
  4. Set category (optional)
  5. Start adding words

Tip: Dividing vocabulary by topics can help you learn more effectively.
